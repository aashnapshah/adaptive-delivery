The problem

The constraint is delivery, not diagnosis

  • LLMs already match or exceed physicians on diagnostic reasoning
  • Goh 2024 (RCT): the model alone beat physicians — but physicians given the model did not improve
  • Indiscriminate alerts → alert fatigue → overrides. A great recommendation, poorly delivered, captures little value

The cost of complexity
